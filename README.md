# Viewcake

An interactive presentation framework.

## Requires

* [NPM](https://www.npmjs.com)
* [Bower](http://bower.io)
* [MongoDB](https://www.mongodb.org)

## Installation

Clone repository then run

    npm install
    bower install

## Start

From the root directory run

    npm start
